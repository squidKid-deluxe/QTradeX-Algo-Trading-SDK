# Molecular Programs

Intermediate complexity programs built from 2-5 atomic operations.

## Categories

- **Perception** - Sensory processing
- **Memory** - Memory systems  
- **Reasoning** - Basic reasoning
- **Attention** - Attention mechanisms
- **Emotion** - Emotional primitives

## Example

```python
from combinatronix.molecules.perception import edge_detector

detector = edge_detector.EdgeDetector()
edges = detector.process(image_field)
```
